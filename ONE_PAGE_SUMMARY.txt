## 🚀 Deploy in 3 Steps

### Step 1: Update + Commit
```powershell
# Update the credentials above, then:
git add .
git commit -m "Production credentials"
```

### Step 2: Push to GitHub
```powershell
git push origin main
```

### Step 3: Connect Cloudflare Pages
1. Go to https://dash.cloudflare.com
2. Click "Pages" → "Create Project" → Select your GitHub repo
3. Build command: `npm run build`
4. Output: `dist/client`
5. Add environment variables:
   - `VITE_PUBLIC_SUPABASE_URL` = your Supabase URL
   - `VITE_PUBLIC_SUPABASE_ANON_KEY` = your Supabase key
6. Deploy!

**Website goes LIVE in ~2 minutes** ⚡
